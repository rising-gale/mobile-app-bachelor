# Vehicle Plate Scanner — Mobile Application

Мобильное приложение на платформе React Native (Expo) для сканирования, распознавания и комплексной оценки автомобильных номеров. Приложение интегрируется с бэкендом на FastAPI и обеспечивает бесшовную работу с нативной камерой устройства.

## 🛠 Технологический стек

- **Framework:** [Expo](https://expo.dev/) (React Native)
- **Language:** TypeScript
- **State Management & API:** Redux Toolkit & [RTK Query](https://redux-toolkit.js.org/rtk-query/overview)
- **Styling:** NativeWind (Tailwind CSS для React Native)
- **Navigation:** Expo Router (File-based routing)
- **Hardware Interaction:** `expo-camera` (модуль CameraView)
- **Secure Storage:** `expo-secure-store`

## 🌟 Основные возможности

1. **Оптимизированный интерфейс камеры (4:3):** Превью камеры жестко синхронизировано с пропорциями физической матрицы сенсора. Это предотвращает искажения (кроп) картинки и гарантирует, что нейросеть на бэкенде получит ровно тот кадр, который пользователь видит на экране.
2. **Автоматическая ротация токенов (JWT Refresh Mutex):** Безопасная аутентификация с использованием механизмов короткоживущих `access_token` и `refresh_token`. Архитектура сетевого слоя защищена атомарной блокировкой (Mutex): при одновременном выполнении нескольких параллельных запросов предотвращается дублирование запросов на обновление сессии.
3. **Безопасное хранение:** Чувствительные данные сессии (токены) изолированы внутри нативного зашифрованного хранилища устройства через `SecureStore`.
